Detailed Description

A class used for doing computations using Sexagesimal math

Sexagesimal math is math using base 60. This is crucial for creating timelines for the USMC because all operations are based upon military time. Therefore, input and output for our application need to be in military time as well. This class keeps track of a single time, with hours, minutes, and seconds represented in military format.
